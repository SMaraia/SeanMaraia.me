Zombie Fiesta

Zombie Fiesta was a project in our Game Development and Algorithmic Problem Solving class. Made in XNA, the game is a roguelike-style game with a randomly generated world. I was responsible for the shader that changed the color of the water, as well as the item creator, found HERE . I built the Interface for the Item Creator from scratch. The application uses JSON.net to export the desired characterestics to an external file, which we then imported into the game proper, creating a new Item with new characteristics. This tool was able to create weapons, consumables, and ammo, and had the capability to be used for armor and stat boosting until time constraints caused those features to be cut from the game.
