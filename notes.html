Notes

I am using the Responsive Grid System by Graham Miller for the basis of my layout, because I like the uniformity of the grids, as well as the flexibility of being able to alter the grid sizing at will. All of the images were created by me, and are of projects that I myself have been a primary or sole contributor to. The script to run the Unity Project came directly from the webplayer.html file that is created when exporting a Unity project. I used the font Exo from Font Squirrel and is licensed for free use, distribution and embedding. The main challenges I encountered were in my use of whitespace, which was always difficult for me to reduce without ruining the page layout on a different screen width, and doing so perfectly felt like it would require a massive amount of additional media queries. In addition, I found it difficult to fill out the content of the site, making the appropriate number of pages. Pages for hobbies, current projects, Tools and Language didn't feel like they were warranted, as the layout would have been far too sparse, so I had to condense that into a single page. It was also difficult to bring in above and beyond work, so I decided to put the page on a personal domain at SeanMaraia.me, which was an interesting challenge in learning how to host my own site.
